# FunGUI

## Description

This repository is where I build fun, creative, and sometimes slightly chaotic GUI experiments.

I like pushing graphical interfaces beyond their "normal" boundaries, for example, creating window overlays that make animations appear as if they're escaping outside the window, interacting with the desktop, or breaking the frame itself.

## Requirements

This package require:

 - python3
 - python3 Standard Library

### Optional

 - Pillow (for pure python image overlay)
